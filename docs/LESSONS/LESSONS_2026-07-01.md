# LESSONS 2026-07-01 · UDN NEWS 懶人包 bodyText + 版型圖生成

## L1：API POST 新欄位不加 = 永遠沒存
- 現象：版型建立時 imageSize 填進去，Firestore 卻空著；description 欄位跑到 "1024x1024"
- 根因：`POST /api/layouts` 只接 name/description/imageUrl，imageSize 沒在型別定義裡，靜默丟棄
- 下次：新增欄位時三個地方同步：類型定義 → API route destructure → firestore 函數簽名
- 對應 feedback：backend-client-must-sync（後端改動必須同步客戶端）

## L2：git root ≠ cwd，git add 帶路徑要引號
- 現象：`git add platform/app/projects/[id]/assets/AssetsClient.tsx` 在 zsh 被 glob 展開報 no matches
- 根因：`[id]` 被 zsh 當 glob 字元，路徑沒加引號
- 下次：含方括號的路徑一律雙引號：`git add "platform/app/projects/[id]/..."`

## L3：同步 Phase A 讓 UI「卡住」是正常現象，但用戶感知不到
- 現象：Adam 說「卡住了」其實是在等 bridge 回文案（30–90 秒）
- 根因：按鈕文字有變但整個區域靜止，視覺反饋不足
- 下次：長同步任務要有更明顯的 loading 指示（spinner、進度文字）
- 待做：評估是否加 progress ring 或換回 fire-and-forget + 輪詢
