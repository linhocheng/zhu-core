# LESSONS 2026-05-17e · Atelier Phase 1 完成

## L1：task_secret 要放 Authorization header 不是 body
- 現象：子代理打 PATCH 一直 Unauthorized，body 帶 task_secret 不行
- 根因：gateway PATCH endpoint 驗證的是 `Authorization: Bearer {task_secret}`，不是 body 欄位
- 下次：子代理 prompt 的 curl 範本一律用 `-H "Authorization: Bearer $SECRET"`
- 對應 feedback：動手前驗證假設（curl 打一次確認格式再寫 prompt）

## L2：aiohttp 不是 gateway 標準庫，用 urllib.request
- 現象：_dispatch_atelier_task 用 aiohttp，gateway 啟動後 Discord 訊息回「No module named aiohttp」
- 根因：gateway 環境沒裝 aiohttp，現有 Atelier 代碼的 aiohttp 都包在 try/except 裡靜默失敗
- 下次：gateway 新代碼 HTTP 請求一律用 urllib.request（stdlib）或 asyncio.to_thread 包裝
- 對應 feedback：靜默失敗用「缺席的 log」診斷

## L3：架構方向轉換前先聊清楚，代碼動得更快
- 現象：本 session 前 30 分鐘聊 Atelier 定位（human-managed vs AI-driven），後面 coding 效率很高
- 根因：搞清楚「Atelier 是 AI 生態不是 project management」後，每個 UI 決策都有依據
- 下次：有大方向轉換的 session，先花 20 分鐘聊清楚北極星，再動手
- 對應 feedback：沒問清楚不開工
