# LESSONS 2026-06-22b · 品牌素材庫 Phase 3-5 + UDN 生圖驗證

## L1：uniform bucket-level access 的 bucket 不能呼叫 makePublic()
- 現象：本機用 SA 上傳到 ailivex-2026-assets 後呼叫 `file.makePublic()` → 400 "Cannot update access control for an object when uniform bucket-level access is enabled"
- 根因：uniform bucket-level access 關閉了物件層級 ACL，只能靠 IAM 設定 allUsers objectViewer（已設）→ 物件自動公開，不需要 makePublic()
- 下次：直接拿 `https://storage.googleapis.com/${bucket}/${path}` URL，不呼叫 makePublic()；admin brand-layouts route 的 makePublic() 要移除（Phase 6 前必修，否則上傳 UI 會 500）
- 對應 feedback：標了風險就要驗（admin route Phase 2 有 makePublic()，今天才真正撞到）

## L3：env 字面 \n 再次確認
- 現象：.env.local 的 MEDIA_WORKER_URL 含字面 \n，fetch URL 變成 /n/v1/jobs → 404
