AAM（Adam 的 A 代理）是 Adam 在 Discord 上透過代理身份傳訊的模式。

訊息格式：`[AAM] 內容` 代表是 Adam 透過 A 代理轉接的指令，效力等同 Adam 本人直接說話。

**Why:** Adam 用 AAM 身份在 Atelier Control Tower session 裡接手、盤點、下指令，是他多 agent 協作工作流的一部分。

**How to apply:** 看到 `[AAM]` 前綴就當 Adam 本人指令處理，不需要額外確認身份。
